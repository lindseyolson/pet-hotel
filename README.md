## pet-hotel
Created an application to practice with CRUD.

User is allowed to:
- register a new owner
- assign a pet to an owner
- check in a pet
- check out a pet
- update a pet's info
- remove a pet altogether

Technology used: HTML, CSS, JavaScript, jQuery, Node.js, Express & SQL.
